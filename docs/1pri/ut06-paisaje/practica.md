## ¡Dibujamos nuestro mundo!

Hoy vamos a ser artistas y observadores de paisajes.

### Reto Inicial: ¿Qué ves?
Asómate a la ventana de clase y di tres cosas que veas:
1. ¿Es algo natural (un árbol, una nube)?
2. ¿Es algo construido (una casa, una farola)?

---

### Paso a Paso: Maqueta de Plastilina
1. Vamos a usar una base de cartón.
2. Con plastilina verde, haremos el campo o las llanuras.
3. Con plastilina marrón, haremos montañas altas.
4. Con plastilina azul, haremos un río que baje por la montaña.

:::info ¡Truco!
Puedes usar palillos o piedrecitas de verdad para que tu paisaje parezca real.
:::

## ¡Misión: Nuestro Herbario Viajero!

¡Enhorabuena! Ya sabes distinguir árboles, arbustos y hierbas.

### El Reto
Vamos a crear un **Herbario de Clase** con hojas de plantas de nuestro entorno.

1. **Recolección**: Busca hojas secas que se hayan caído de los árboles o plantas del patio o de tu calle.
2. **Secado**: Ponlas entre las hojas de un libro viejo durante unos días para que se queden planas.
3. **Ficha**: Pega la hoja en una cartulina y escribe:
   - Nombre de la planta.
   - ¿Es árbol, arbusto o hierba?
   - Lugar donde la encontraste.
4. **Unión**: Juntaremos todas las cartulinas para formar un gran libro de plantas.

:::tip ¡Misión Cumplida!
Al crear este herbario, estamos aprendiendo a valorar la biodiversidad que nos rodea. ¡Excelente trabajo!
:::
